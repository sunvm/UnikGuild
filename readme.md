# UnikGuild - Плагин для управления гильдиями в Minecraft

## Описание
UnikGuild - это плагин для серверов Minecraft (Paper 1.21.11), который позволяет игрокам создавать и управлять гильдиями через интерактивную систему с использованием предметов.

## Основные возможности

### Система гильдий
- Создание гильдий через золотой меч
- Лимит участников: 15 человек на гильдию
- Уникальные аббревиатуры (3 символа) для каждой гильдии
- Цветовые схемы для визуального различения гильдий

### Управление предметами
- **Золотой меч** - создание гильдий и приглашение участников
- **Палка изгнания** - исключение участников из гильдии (ЛКМ по игроку)
- Автоматическое удаление предметов при удалении гильдии

### Социальные функции
- Чат гильдий (префикс `- `)
- Информация о гильдиях через команды
- Отображение аббревиатур в TAB-листе
- Система приглашений с подтверждением

### Административные функции
- Просмотр всех гильдий на сервере
- Удаление гильдий
- Выдача предметов гильдий

## Установка

### Требования
- Сервер: Paper/Spigot 1.21.11
- Java: версия 21
- Права: стандартные разрешения Minecraft

### Шаги установки
1. Скачайте файл `UnikGuild-1.0.0.jar`
2. Поместите файл в папку `plugins/` вашего сервера
3. Перезапустите сервер
4. Плагин автоматически создаст необходимые конфигурационные файлы

## Использование

### Создание гильдии
1. Получите золотой меч (через творческий режим или команду `/guilds give`)
2. Нажмите ПКМ с мечом в руке
3. Введите название гильдии в чат (2-16 символов)
4. Выберите цвет гильдии (цифра 1-8)
5. Введите аббревиатуру (3 символа, буквы/цифры)

### Приглашение участников
1. Основатель гильдии ударяет игрока золотым мечом (ЛКМ)
2. Игрок получает запрос на вступление
3. Игрок использует команду `/g accept` для принятия приглашения
4. Меч теряет 1 использование (всего 15 использований)

### Команды для игроков

| Команда | Описание | Пример |
|---------|----------|---------|
| `/g accept` | Принять приглашение в гильдию | `/g accept` |
| `/g deny` | Отклонить приглашение | `/g deny` |
| `/g members` | Показать участников гильдии | `/g members` |
| `/g kick <ник>` | Исключить игрока из гильдии | `/g kick Player1` |
| `/g leave` | Покинуть гильдию | `/g leave` |
| `/g info` | Информация о гильдии | `/g info` |

### Административные команды

| Команда | Описание | Разрешение |
|---------|----------|------------|
| `/guilds list` | Список всех гильдий | `unikguild.admin` |
| `/guilds delete <название>` | Удалить гильдию | `unikguild.admin` |
| `/guilds give <ник>` | Выдать меч гильдии | `unikguild.admin` |

### Чат гильдии
Для общения внутри гильдии используйте префикс `-`:
"- Привет, гильдия!"
Это сообщение увидят только участники вашей гильдии.

  default: true

unikguild.admin:
  description: Административные права гильдий
  default: op

Конфигурация

Плагин автоматически создает файл конфигурации plugins/UnikGuild/guilds.yml для сохранения данных гильдий.
Ограничения

    Максимальное количество участников в гильдии: 15

    Минимальная длина названия гильдии: 2 символа

    Максимальная длина названия гильдии: 16 символов

    Длина аббревиатуры: ровно 3 символа

Устранение проблем
Частые проблемы

    Плагин не загружается

        Проверьте версию Java (должна быть 21)

        Проверьте версию Paper (должна быть 1.21.11)

        Проверьте лог сервера на наличие ошибок

    Команды не работают

        Убедитесь, что у вас есть необходимые разрешения

        Проверьте правильность синтаксиса команд

        Перезагрузите плагин командой /reload confirm

Логирование

Плагин записывает информацию в:

    logs/latest.log - общие логи сервера

    plugins/UnikGuild/guilds.yml - данные гильдий

    Консоль сервера - информация о загрузке плагина
